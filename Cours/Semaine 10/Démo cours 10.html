Carte web — Exemple complet à déconstruire

Objectifs : afficher une carte, ajouter un titre, charger un GeoJSON local, styliser selon un attribut, ajouter un filtre, puis publier.

Contrôles

Fichier attendu :
./data/territoires.geojson

Le GeoJSON doit idéalement contenir un attribut category pour le style et le filtre, ainsi qu’un attribut name pour le popup.

Si le fichier n’est pas trouvé, un jeu de données de démonstration sera utilisé automatiquement.
Légende
Publier :
1. Déposer ce fichier comme index.html
2. Ajouter le dossier data/
3. Publier sur GitHub Pages, Netlify ou Vercel
